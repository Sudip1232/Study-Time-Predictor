# 🚀 How to Run the Web App

## Step 1: Install Dependencies

```bash
pip3 install flask pandas numpy scikit-learn joblib matplotlib
```

## Step 2: Start the Web Server

```bash
python3 app.py
```

## Step 3: Open in Browser

Go to: **http://localhost:5000**

## Features

✅ **Upload & Train**: Upload CSV dataset and train ML model  
✅ **Statistics Dashboard**: View dataset statistics with charts  
✅ **Make Predictions**: Enter student info and get predictions  
✅ **Visual Graphs**: See comparison charts and genre distribution  
✅ **Neighbor Analysis**: View similar students from training data  

## What You'll See

1. **Training Section**:
   - Upload CSV file
   - See training progress
   - View dataset statistics (samples, averages)
   - Genre distribution bar chart

2. **Prediction Section**:
   - Enter screen time, book genre, gaming hours
   - Get prediction with visual comparison chart
   - See 5 nearest neighbors table

Use the included `Test Data.csv` file to test the system!
